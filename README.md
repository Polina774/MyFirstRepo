🚀 Мой репозиторий
📄 Описание репозитория
Этот репозиторий создан для изучения GitHub, работы с feature-ветками, системой контроля версий и расширениями GitHub Marketplace. Здесь выполняются задания по созданию веток, внесению изменений и объединению их с основной веткой main через pull request.
📋 Список задач и возможностей
1. Работа с репозиториями
   - создание файлов
   - редактирование кода
2. Использование feature-веток
   - создание отдельных веток для задач
   - слияние через pull request
3. Контроль версий Git
   - отслеживание изменений
   - работа с историей коммитов
4. GitHub Marketplace
   - установка расширений
   - использование Dependabot
5. Совместная работа
   - code review
   - работа в команде
